# student-sql-analytics
SQL-only student analytics project demonstrating joins, aggregations, and data analysis. 
## Dataset
The dataset contains three tables:
- students
- courses
- enrollments

## Concepts Covered
- SELECT, WHERE
- JOINs
- GROUP BY and HAVING
- Aggregations (COUNT, AVG, MAX)
- CASE statements
- NULL handling

## Files
- schema.sql – table definitions
- queries.sql – analytical SQL queries

## Notes
This is a SQL-only project. No Pandas or Python is used.
